## Final design goal
Create a polished desktop website design based on the approved wireframe.

The final design must improve:

- visual hierarchy
- readability
- B2B credibility
- component consistency
- content scanning
- spacing rhythm
- CTA clarity
- professional presentation

---

## Brand style direction
The design should feel:

- modern
- clean
- corporate
- technical
- trustworthy
- engineering-focused
- suitable for B2B electronics industry

Avoid:

- consumer-style design
- childish visuals
- too many colors
- decorative clutter
- long content blocks
- copied competitor style
- fake product claims

---

## Typography
Use:

- heading font: `Poppins`
- body font: `Inter`

Keep typography consistent.

Use clear difference between:

- page title
- section heading
- card title
- body text
- caption
- button text
- label

---

## Layout
Default desktop frame:

- frame width: `1440 px`
- content width: `1200 px` to `1280 px`
- content centered

Use a clean grid system.

Use generous but controlled whitespace.

---

## Spacing
Use 8-point spacing system.

Preferred spacing:

- 4
- 8
- 12
- 16
- 24
- 32
- 40
- 48
- 64
- 80
- 96

Recommended desktop section spacing:

- tight: `48 px`
- normal: `64 px`
- spacious: `80 px`
- hero / major section: `96 px`

---

## Components
Use reusable components from:

`02 Components / Design System`

Use or create consistent components for:

- header
- footer
- buttons
- cards
- product cards
- application cards
- tabs
- accordions
- CTA sections
- forms
- tables
- slider controls
- section headings

Do not detach or restructure components without approval.

---

## Image rules
Use real images only when provided and approved.

If images are missing:

- use clean placeholders
- label placeholders clearly
- keep ratio consistent
- do not invent fake technical images

---

## Content rules
Copy must be:

- concise
- scannable
- B2B
- practical
- technical but readable

Do not:

- copy competitor text
- invent product specifications
- overuse marketing hype
- create duplicated sections
- create very long paragraphs
